# Glossary

Use this page when a docs page mentions a package term and you want the short definition.

## Common terms

- `Dictionary`: the runtime translation object
- `dictionaryFactory`: the app-supplied factory that builds your generated dictionary type
- `previewDictionaries`: in-memory locale maps used before assets
- `fallbackLocale`: the locale used when a key or locale is unavailable
- `catalog`: the separate translation management UI and API
- `profile`: the validation strictness preset
- `placeholder schema`: optional type/required metadata used during validation

## Quick reminder

```text
Dictionary = runtime values
Generator = typed API
Validator = consistency checks
Catalog = sidecar management UI/API
```
