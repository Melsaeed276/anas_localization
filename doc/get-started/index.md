# Get Started

Use this section when you want the shortest route from package install to a translated Flutter screen.

## What this section covers

- install the package
- add locale files
- generate the dictionary
- wrap your app with `AnasLocalization`

## Quick path

```bash
flutter pub add anas_localization
dart run anas_localization:localization_gen
dart run anas_localization:anas_cli validate assets/lang
```
