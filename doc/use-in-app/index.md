# Use in App

Use this section after the package is installed and your app already boots with `AnasLocalization`.

## What this section covers

- reading generated translations
- changing locale at runtime
- preview dictionaries for tests and previews
- custom translation loaders and fallback behavior

## Quick example

```dart
final dict = getDictionary();
Text(dict.appName);
```
